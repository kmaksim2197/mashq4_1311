# Parol Boshqaruv Tizimi

Xavfsiz foydalanuvchi autentifikatsiya tizimi (OOP tamoyillari asosida).

## Xususiyatlar
- Parol **maxfiy** (`__password`)
- `set_password()` — kamida **8 belgi**
- Parolni faqat `AuthenticationSystem` tekshiradi
- Foydalanuvchi parolni **o‘qiy olmaydi**
